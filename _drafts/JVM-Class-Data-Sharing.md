## 前言

​	最初的 **Class data sharing (CDS)** 是 **J2SE 5.0** 新增的特性，主要是为了降低应用的启动时间，同时也能够减少内存的使用。它仅支持 **Java HotSpot Client VM**，垃圾收集器也仅支持 **serial garbage collector**。这样的局限性，导致该特性早期并没有被广泛使用。
